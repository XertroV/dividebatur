Introduction

This site presents the details results for Single Transferable Vote elections to fill Australian Senate vacancies. All counts are shown. The particular variant of STV implemented is that specified under the Commonwealth Electoral Act (1918).

Both above-the-line and below-the-line votes are included in the count.

The results presented on this site are believed to precisely match those produced by the Australian Electoral Commission's software. Where detailed run logs have been obtained from the AEC, the totals for each candidate (and the number of exhausted/gain-loss votes) have matched at each count, and the candidates elected and excluded have matched. If you spot something incorrect, please get in touch.

This system is named dividebatur and is open source software distributed under the Apache License, 2.0. It was written out of curiosity, and to check whether the AEC's results are reproducable. It also allows a full STV vote distribution for hypothetical situations, which can be very useful – particuarly when West Australian Senate elections go awry!
